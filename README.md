# Cybersecurity Portfolio
Built by Aakash Aryal | Aspiring Cybersecurity Engineer

## About
This portfolio contains three cybersecurity tools built
from scratch in Python using Kali Linux.

---

## Project 1 — Network Packet Analyzer
A real-time packet capture tool built with Scapy.

**Features:**
- Captures live network packets
- Reads source IP, destination IP, port, protocol
- Applies firewall rules to block suspicious traffic
- Logs all results with timestamps to CSV

**Skills:** Python, Scapy, TCP/IP, Networking

---

## Project 2 — Firewall
A rule-based firewall that inspects live network traffic.

**Features:**
- Blocks traffic by IP, port, and protocol
- Real-time packet inspection
- Colour coded terminal output
- Logs every decision with timestamp to CSV

**Skills:** Python, Scapy, Network Security, Logging

---

## Project 3 — Cryptography Toolkit
A collection of encryption tools built from scratch.

**Features:**
- Caesar cipher with encrypt and decrypt
- Vigenere cipher with keyword encryption
- RSA public and private key encryption
- AES file encryption

**Skills:** Python, Cryptography, RSA, AES, Encryption

---

## How to Run

Install dependencies:
pip install scapy pycryptodome

Run packet analyzer:
sudo python3 packet_analyzer.py

Run firewall:
sudo python3 firewall.py

Run cryptography tools:
python3 caesar_cipher.py
python3 vigenere_cipher.py
python3 rsa_encryption.py
python3 aes_encryption.py
