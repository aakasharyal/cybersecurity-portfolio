# Packet Analyzer
A real-time network packet analyzer built in Python using Scapy.

## What it does
- Captures live network packets
- Reads source IP, destination IP, port, protocol
- Applies firewall rules to block suspicious traffic
- Logs all results with timestamps to packet_log.csv

## How to run
sudo python3 packet_analyzer.py

## Skills used
- Python, Scapy
- Networking: TCP, UDP, ICMP, DNS, DHCP
- CSV file logging
- Firewall logic
